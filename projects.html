Advanced To-Do List Application

March 2025 - Present In Progress
Live Demo View Code

Project Overview

A comprehensive task management application designed to help users organize their daily activities with advanced features like reminders, categorization, and cross-platform synchronization.

The application is being developed using Flutter for a seamless cross-platform experience (iOS, Android, Web) with Firebase handling the backend services including authentication, database, and cloud functions.

Key Features

  • User authentication (email/password, Google Sign-In)
  • Task creation with due dates, priorities, and categories
  • Push notifications for reminders
  • Offline functionality with sync when online
  • Dark/Light theme support
  • Data visualization with completion statistics

Technologies Used

Flutter Dart Firebase Firestore Cloud Functions

Challenges & Solutions

Challenge: Implementing real-time synchronization across multiple devices while maintaining offline functionality.

Solution: Utilized Firebase Firestore with cache persistence enabled, combined with a custom sync manager that queues changes when offline and applies them when connectivity is restored.

Library Management System

February 2025 Completed
Live Demo View Code

Project Overview

A desktop application for managing library operations including book inventory, patron records, checkouts, and returns. Developed as a Java application with Swing GUI and MySQL database.

The system was designed to streamline library operations, reduce manual work, and provide librarians with an efficient tool for managing their collections and patron interactions.

Key Features

  • Book inventory management (add, edit, delete, search)
  • Patron registration and management
  • Checkout/return system with due date tracking
  • Late fee calculation
  • Report generation (popular books, overdue items)
  • Multi-user login with different permission levels

Technologies Used

Java Swing MySQL JDBC MVC Pattern

Challenges & Solutions

Challenge: Designing a database schema that could efficiently handle complex relationships between books, copies, patrons, and transactions.

Solution: Implemented a normalized database structure with proper indexing and foreign key relationships. Created optimized SQL queries and used connection pooling to improve performance.

Urban Homes Real Estate

January 2024 Completed
Live Demo View Code

Project Overview

A responsive real estate website that allows users to browse property listings, filter search results, and contact agents. Built with pure HTML, CSS, and JavaScript as a frontend project.

The website features a clean, modern design with intuitive navigation and property showcases designed to highlight the best features of each listing.

Key Features

  • Responsive design for all device sizes
  • Property search with multiple filters
  • Interactive property galleries
  • Agent contact forms
  • Featured listings carousel
  • Neighborhood information sections

Technologies Used

HTML5 CSS3 JavaScript Responsive Design CSS Grid/Flexbox

Challenges & Solutions

Challenge: Creating a responsive design that maintained usability and visual appeal across all device sizes from desktop to mobile.

Solution: Implemented a mobile-first approach using CSS Grid and Flexbox for layout, with careful media query breakpoints to ensure optimal viewing at every screen size.

User Authentication System

Upcoming Planned

Project Overview

A secure user authentication system that will implement modern security best practices including password hashing, CSRF protection, and JWT authentication.

The system will be built as a microservice that can be integrated with other applications, providing a robust foundation for user management and access control.

Planned Features

  • Email/password authentication
  • Social login (Google, Facebook)
  • Email verification
  • Password reset flow
  • Role-based access control
  • JWT token authentication
  • Rate limiting and security headers

Technologies Planned

Node.js Express MongoDB JWT OAuth

Expected Challenges

Challenge: Implementing secure authentication flows that protect against common vulnerabilities like session hijacking, CSRF, and brute force attacks.

Solution Approach: Will implement industry-standard security practices including proper password hashing with bcrypt, CSRF tokens, secure cookie settings, and rate limiting on authentication endpoints.
